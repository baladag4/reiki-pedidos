📦 INSTALAÇÃO EM 3 PASSOS SIMPLES:

═══════════════════════════════════════════════════════════════

PASSO 1 - COLOCAR NO SERVIDOR
─────────────────────────────────────────────────────────────

📁 Copie a pasta "sistema-reiki" para dentro de "htdocs"

Caminho final: C:\xampp\htdocs\sistema-reiki\

⚠️ IMPORTANTE: A pasta deve ficar DIRETO no htdocs, não dentro 
   de outra pasta!

   ✅ CERTO:   C:\xampp\htdocs\sistema-reiki\
   ❌ ERRADO:  C:\xampp\htdocs\radar2\sistema-reiki\
   ❌ ERRADO:  C:\xampp\htdocs\meu-site\sistema-reiki\


═══════════════════════════════════════════════════════════════

PASSO 2 - IMPORTAR BANCO DE DADOS
─────────────────────────────────────────────────────────────

1. Abra o phpMyAdmin:
   👉 http://localhost/phpmyadmin

2. Clique em "Importar" no topo

3. Clique em "Escolher arquivo"

4. Selecione: sistema-reiki/database.sql

5. Clique em "Executar" no final da página

✅ PRONTO! Banco criado com sucesso!


═══════════════════════════════════════════════════════════════

PASSO 3 - ACESSAR O SISTEMA
─────────────────────────────────────────────────────────────

🌐 PÁGINA PÚBLICA (onde usuários enviam pedidos):
   http://localhost/sistema-reiki/

🔐 ÁREA ADMINISTRATIVA:
   http://localhost/sistema-reiki/admin/login.php

   Login: admin@reiki.com
   Senha: admin123

🔧 CONFIGURAÇÕES (opcional)
─────────────────────────────────────────────────────────────

Se o banco de dados tiver usuário/senha diferente de root:

1. Abra o arquivo: sistema-reiki/config/database.php

2. Altere estas linhas:

   define('DB_USER', 'root');        ← Seu usuário aqui
   define('DB_PASS', '');            ← Sua senha aqui

3. Salve o arquivo

🆘 PROBLEMAS?
─────────────────────────────────────────────────────────────

❌ "Forbidden" ou "Acesso Negado"
   → Verifique se a pasta está DIRETO no htdocs
   → Reinicie o Apache no XAMPP

❌ "Erro de conexão com banco"
   → Verifique se importou o database.sql
   → Confira config/database.php

❌ Página em branco
   → Verifique se Apache e MySQL estão rodando no XAMPP
   → Aperte F5 para recarregar
